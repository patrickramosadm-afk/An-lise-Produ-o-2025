# 📊 Análise de Produção – Unidade CAPSi | 2025

Este projeto apresenta um dashboard desenvolvido no Power BI com o objetivo de analisar a produção anual de atendimentos de uma unidade CAPSi.

A proposta vai além da visualização de dados, buscando transformar informações operacionais em insights que apoiem a tomada de decisão.

---

## 🎯 Objetivo

Analisar o volume de atendimentos realizados ao longo do ano, identificando padrões, distribuição por categoria profissional e principais tipos de procedimentos.

---

## 📁 Base de Dados

Os dados utilizados representam atendimentos realizados na unidade, contendo informações como:

- Data do atendimento
- Profissional
- Categoria profissional
- ID do paciente
- Sexo
- Faixa etária
- Status (incluindo busca ativa)

*Obs: Dados tratados e organizados para fins de análise.*

---

## 📊 Principais Análises

O dashboard permite visualizar:

- 📈 Evolução dos atendimentos ao longo dos meses
- 👥 Total de pacientes atendidos
- 🔎 Pacientes em busca ativa
- 👨‍⚕️ Distribuição por categoria profissional
- 🧾 Principais procedimentos realizados
- 🧠 Identificação de padrões e períodos de maior demanda

---

## 💡 Insights Obtidos

- Setembro e outubro concentram o maior volume de atendimentos
- Educadores e psicólogos representam grande parte da produção
- Média de atendimentos por paciente indica recorrência nos acompanhamentos

---
